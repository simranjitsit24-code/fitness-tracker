# 💪 Fitness Tracker

A full-stack fitness tracking application built with React, Node.js, Express, and MongoDB. Track your workouts, monitor your progress, and stay motivated with real-time analytics and visualizations.

## ✨ Features

### 🔐 Authentication
- Secure user registration and login
- JWT-based authentication
- Password validation with strength indicator
- Protected routes for authenticated users

### 📊 Dashboard
- **Workout Statistics**: Total workouts, duration, average rating, and current streak
- **Progress Tracking**: Weekly progress vs previous week
- **Data Visualization**: Interactive charts showing workout trends and distribution
- **Best Workout Type**: Identify your most frequent workout type

### 📝 Workout Management
- **Create Workouts**: Add workouts with type, duration, intensity, and exercises
- **Edit Workouts**: Update existing workout details
- **Delete Workouts**: Remove workouts you no longer need
- **Exercise Tracking**: Add multiple exercises per workout with sets and reps
- **Rating System**: Rate each workout from 1-5 stars

### 🎨 Modern UI
- Dark theme with gradient effects
- Responsive design for all screen sizes
- Mobile-friendly with bottom navigation
- Real-time data updates
- Interactive charts using Chart.js

## 🛠️ Tech Stack

### Frontend
- **React 18** - UI Library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router v6** - Navigation
- **Axios** - API requests
- **Chart.js** - Data visualization
- **React Hot Toast** - Notifications

### Backend
- **Node.js** - Runtime
- **Express** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
- **Bcrypt.js** - Password hashing
